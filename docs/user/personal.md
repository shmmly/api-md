# 个人中心

## 个人信息接口

**功能：**
返回用户的一些个人信息：用户名、头像等

- 提供参数：用户id
- 返回参数：用户的信息


## 用户出行总数和里程数接口
**功能：**
返回用户所有的出行总次数、以及总的里程数

- 提供参数：用户id
- 返回参数：出行次数、总里程数

::: tip
    里程数没有
:::

## 用户评价的个数以及未评价个数
**功能：**
获取用户已经评价和未评价的个数

- 提供参数：用户id
- 返回参数：评价个数、未评价次数

## 修改用户昵称接口
**功能：**
可以修改用户的昵称
- 提供参数：用户id 修改之后的名称
- 返回参数：是否成功

## 用户密码修改接口
**功能：**
修改用户的密码
- 提供参数：用户id 新密码
- 返回参数： 是否成功

## 获取用户所有评价接口
**功能：**
- 提供参数：用户id
- 返回参数：用户的评价带分页按时间排序

## 获取用户评价分类个数的接口
**功能：**
可以获取用户全部评价、好评、中评、差评的个数
- 提供参数：用户id
- 返回参数：
:::error
暂无此功能
:::

可以返回用户所有去过的地方的经纬度等用来在地图上面标记。
以及可以返回所去过的市级城市例如泰州、扬州等
- 提供参数：用户id
- 返回参数：

::: danger
    无经纬度数据，需网关支撑，后续版本再议
:::
